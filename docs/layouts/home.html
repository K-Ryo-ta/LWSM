ls に検索を。
多すぎるフォルダの中身を一瞬で絞り込む。

LWSM は、ls コマンドでフォルダの中身を見るときに、中身が多すぎる場合でも 単語検索・文章検索・内容検索で 目的のファイル・ディレクトリを素早く見つけられ、文字数カウントも行える コマンドラインツールです。

ドキュメントを見る ソースコードを見る
zsh — lwsm

$ ls

main.rs lib.rs config.rs entries.rs search.rs output.rs
Cargo.toml Cargo.lock README.md LICENSE tests/ target/ ...

$ lwsm -m rust

gamma-rust.txt

$ lwsm -s "hello world"

hello_world.txt

$ lwsm -c "TODO"

main.rs

$ lwsm -t README.md

2688 README.md

$ ▋

## 特徴

「探す」をシンプルに。ls の延長で使える検索コマンド。

-m

単語マッチ検索

ファイル名・ディレクトリ名を単語単位で完全一致検索。_ や -、. も単語区切りとして扱うので、my-rust-lib も「rust」でヒットします。

-s

文章検索

部分一致の文章検索。大文字・小文字を区別しないので、README.md も「readme」で見つかります。

-c

内容検索

ファイル名ではなく中身を読んでキーワードを検索。大文字・小文字を区別せず、"TODO" を含むファイルだけを一覧できます。

-t

文字数カウント

ファイル、またはディレクトリ内の各ファイルの文字数を集計。ディレクトリ指定なら各ファイルの文字数と合計（total）をまとめて表示します。空白・改行を除き、日本語も1文字として数えます。

⚡

Rust 製で高速・軽量

Rust で実装された軽量な単一バイナリ。依存クレートなしで動作し、macOS / Linux / Windows をサポートします。

## クイックスタート

引数はモード・クエリ・パス（省略時はカレントディレクトリ）の3つだけ。

コマンド形式

$ lwsm -m <word> [path]
$ lwsm -s <sentence> [path]
$ lwsm -c <keyword> [path]
$ lwsm -t [path]

実行例

$ lwsm -m rust
$ lwsm -s "hello world"
$ lwsm -c "TODO" ./src
$ lwsm -t ./src
